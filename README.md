# Problem Solving Patterns

## Frequency Counter

Use objects or maps to count values and compare frequencies efficiently.

Example problems:

- Same Frequency
- Valid Anagram

Time complexity improvement example:

Brute Force
O(n²)

Frequency Counter
O(n)

---

## Multiple Pointers

Use two pointers moving through a data structure to reduce complexity.

Example problems:

- Count Unique Values
- Is Subsequence

---

## Sliding Window

Efficiently compute results over a subset of data.

Example problems:

- Maximum Subarray Sum
- Longest Substring Without Repeating Characters

---

## Recursion

A function that calls itself until a base condition is reached.

Example problems:

- Factorial
- Fibonacci
- Flatten Array
